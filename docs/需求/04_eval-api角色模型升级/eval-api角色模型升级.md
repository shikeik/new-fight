# eval-api 角色模型升级

**描述**：通过 eval-api 对游戏内的角色模型进行逐步升级和美化。以脚本注入的方式实时调整模型结构、材质、光照或特效，实现“一笔一画”的迭代优化。

**工作方式**：
- 所有调整先通过 eval-api 在运行中的游戏里实时试验。
- 效果确认后再同步到源码中做持久化。
- 便于在真机浏览器上即时预览修改结果。

**预期方向（待细化）**：
- [ ] 角色身体比例优化
- [ ] 材质与配色升级
- [ ] 关节与动画细节增强
- [ ] 光影与自发光效果
- [ ] 武器/服装等配件建模

**状态**：规划中，等待统一入口和第一批具体修改项。
